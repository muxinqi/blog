macOS常用软件清单

July 04, 2019

铺垫

在 macOS Catalina Public Beta 的使用过程中，遇到了一些软件闪退的问题，就比如正在使用 Typora 写文章的时候，一个猝不及防的闪退之后，重现打开软件发现一个字都没有自动保存是多么的绝望……

于是手动降级原来的正式版，失败。😭

就成这样了，旁边没有 macOS 系统的设备，网上 Windows 制作安装盘的教程又有些风险，于是只能求助 Apple 售后，电话预约的时间太晚了，要等将近一个星期了。

就自己在 Apple 支持 上预约了一个当天就有空的售后点，去了之后给我的感觉，店这么大怎么这么空呢……

详细问了一下才知道，过保 MacBook 重装系统是收费 ¥300 的（除 Apple Store 外）。

¥300 对于一个从小到大刷机无数的学生来说不算什么（其实很在意，很贵了好吧）

于是直接奔向最近的 Apple Store，虽然手机上显示已经没有时间可以预约了，不过跟小哥哥说一说还是可以加号的。

接着去吃了个饭，回来时间也差不多了。

（外婆家的酒酿圆子羹，第一眼以为里面红色的是番茄，后来发现竟然是==西瓜==，真心爱了！）

就看小哥哥三下五除二，不超过半小时，就把电脑恢复了原样。

由于没有备份习惯的我（本有机会 也没备份），面对一个崭新的电脑无从下手，软件只能想起一个是一个，所以有了写这个清单的打算。


List

Browser

  • Chrome - “心灵的窗户”
  • Microsoft Edge - 目前主力

Editor

  • Sublime Text 3 - 文本编辑器，快！
  • Visual Studio Code - 编辑器里面的 Chrome，插件丰富。
  • Coderunner 3 - 平常跑课上的代码片段会用到，支持调试，方便！
  • Typora - Markdown编辑器，在这里你能看到的文章就都是用它写的
  • WPS Office for Mac

IM

  • Wechat - 国内“毒瘤”
  • QQ - “毒瘤他爹”
  • Telegram - “安全，快”

IDE

  • JetBrains Toolbox - 管理下面这几个软件的
  • PyCharm
  • IntelliJ IDEA

Media

  • IINA - 媒体播放|简约强大|开源免费
  • NetEase Music - 平常放背景音乐的家伙

Efficiency

  • iTerm 2 - 终端，从未关过的软件之一。
  • eZip - 专为macOS所设计的解压缩软件
  • Free Download Manager
  • iStat Menus - 显示系统详细的状态信息
  • NTFS for Mac - 读写NTFS格式外部存储
  • iTerm 2 - 比自带Terminals好看还好用
  • Alfred 4
  • uTools - 因为还没有购买Alfred 4刚好发现了这个 在试用中
  • LastPass - 密码管理 免费足够用了
  • Magnet - 广告语“使您的工作空间井然有序” 非常值得入手
  • CheatSheet - 忘了快捷键？长按 Command 启动
  • Pap.er - 适合高分屏的壁纸软件 精品
  • Proxifier - 配合下面这个软件用
  • balenaEtcher - 给U盘烧写系统镜像，堪比Win上的Rufus～
  • ShadowsocksX-NG-R8 - 不用多说的好伙伴

CLI

  • Homebrew - macOS 上强大的包管理工具

安装命令

记录一下，以备不时之需。

brew

brew install git gnupg htop lrzsz

brew cask
